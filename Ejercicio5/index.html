📝 Longest Word Finder

Using the Sliding Window technique to find the longest word in your text.

Result:

Click the button to see the longest word

💡 How it works (Sliding Window technique):

  1. Split the text into an array of words using .split(' ')
  2. Initialize a window that slides through each word
  3. Compare each word's length with the current longest word
  4. Update the result when a longer word is found
  5. Return the longest word at the end
