🎯 Two Pointers Algorithm

Find consecutive guests with names starting with the same letter

📖 How It Works

👇 Left Pointer (start) Points to current guest
➡️
👇 Right Pointer (next) Points to next guest

1️⃣ Start with pointers at positions 0 and 1
2️⃣ Compare first letters of both names
3️⃣ If they match → Found a pair! 🎉
4️⃣ If not → Move both pointers forward
5️⃣ Repeat until pair is found or end of list

💻 Algorithm Code


function encontrarPareja(arr) {
    let inicio = 0;
    let siguiente = 1;
    
    while (siguiente < arr.length) {
        // Get first letters of both names
        const letra1 = arr[inicio][0].toUpperCase();
        const letra2 = arr[siguiente][0].toUpperCase();
        
        // Compare initials
        if (letra1 === letra2) {
            return [arr[inicio], arr[siguiente]];
        }
        
        // Move both pointers forward
        inicio++;
        siguiente++;
    }
    
    return null;
}

🎯 Two Pointers Algorithm | Time Complexity: O(n) | Space Complexity: O(1)
